# PsicoResponde CyL

Prototipo web para coordinar psicologos voluntarios en situaciones de emergencia en Castilla y Leon.

## Resumen

El proyecto plantea una interfaz de gestion para registrar profesionales, visualizar disponibilidad y ordenar la respuesta ante incidencias. Esta orientado a mostrar criterio funcional, estructuracion de pantallas y entendimiento de procesos de coordinacion.

## Mi aportacion

- Definicion de flujo funcional para registro, consulta y coordinacion.
- Diseno de interfaz orientada a claridad y respuesta rapida.
- Estructuracion de datos visibles para perfiles, disponibilidad e incidencias.
- Prototipado web estatico para validar la idea antes de una implementacion completa.

## Funcionalidades

- Pantalla de acceso y formulario de perfil.
- Panel de disponibilidad y estado de profesionales.
- Visualizacion de informacion operativa para coordinacion.
- Estilos responsive para escritorio y movil.

## Stack

- HTML.
- CSS.
- JavaScript en cliente.
- Librerias externas para iconos, graficas y PDF.

## Estado

Prototipo de portfolio. No es una aplicacion lista para uso real en emergencias ni contiene datos personales reales. Cualquier uso operativo requeriria backend seguro, control de acceso, proteccion de datos, trazabilidad y validacion con especialistas.
